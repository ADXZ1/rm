GUI_0 机械臂操控系统使用文档

系统概述

GUI_0 是一个机械臂操控系统界面，提供机械臂轨迹控制、末端工具控制、锁枪控制、继电器控制、底座电机控制和机械臂配置等功能。系统界面分为三个主要区域：左侧面板（机械臂控制）、中间面板（实时相机预览）、右侧面板（工具和配置控制）。

界面布局

系统界面采用三列布局：
- 左侧面板：机械臂轨迹控制
- 中间面板：实时相机预览（深度相机）
- 右侧面板：末端工具控制、锁枪控制、继电器控制、底座电机控制、机械臂配置

功能分类说明

一、轨迹控制（左侧面板）

此区域用于控制机械臂的轨迹录制和回放功能。

1. 录制轨迹
- 按钮名称：录制轨迹
- 对应脚本：Path/Record_Path.py
- 功能说明：开始录制机械臂的运动轨迹，保存为轨迹文件

2. 回轨迹起点
- 按钮名称：回轨迹起点
- 对应脚本：Path/Path_Begin.py
- 功能说明：将机械臂移动到已录制轨迹的起始位置

二、末端工具控制（右侧面板）

此区域用于控制机械臂末端执行器的基本操作。

1. 打开夹爪
- 按钮名称：打开夹爪
- 对应脚本：release.py
- 功能说明：控制机械臂夹爪打开，释放物体

2. 合上夹爪
- 按钮名称：合上夹爪
- 对应脚本：grab.py
- 功能说明：控制机械臂夹爪闭合，抓取物体

3. 快换手解锁
- 按钮名称：快换手解锁
- 对应脚本：Kuaihuanshou_unlock.py
- 功能说明：执行快换手解锁操作，释放当前工具

4. 快换手上锁
- 按钮名称：快换手上锁
- 对应脚本：Kuaihuanshou_lock.py
- 功能说明：执行快换手上锁操作，锁定当前工具

三、锁枪控制（右侧面板）

此区域用于控制锁枪工具的取用和放回操作。

1. 取用Y1
- 按钮名称：取用Y1
- 对应脚本：YIYEQIANG_QU_Y1.py
- 功能说明：从工具库中取用Y1锁枪工具

2. 放回Y1
- 按钮名称：放回Y1
- 对应脚本：YIYEQIANG_FANG_Y1.py
- 功能说明：将Y1锁枪工具放回工具库

3. 取用Y2
- 按钮名称：取用Y2
- 对应脚本：YIYEQIANG_QU_Y2.py
- 功能说明：从工具库中取用Y2锁枪工具

4. 放回Y2
- 按钮名称：放回Y2
- 对应脚本：YIYEQIANG_FANG_Y2.py
- 功能说明：将Y2锁枪工具放回工具库

四、继电器控制（右侧面板）

此区域用于控制继电器的锁紧和解锁操作。

1. Y1锁紧
- 按钮名称：Y1锁紧
- 对应脚本：Y1_SUO.py
- 功能说明：控制Y1继电器执行锁紧操作

2. Y1解锁
- 按钮名称：Y1解锁
- 对应脚本：Y1_JIE.py
- 功能说明：控制Y1继电器执行解锁操作

3. Y2锁紧
- 按钮名称：Y2锁紧
- 对应脚本：Y2_SUO.py
- 功能说明：控制Y2继电器执行锁紧操作

4. Y2解锁
- 按钮名称：Y2解锁
- 对应脚本：Y2_JIE.py
- 功能说明：控制Y2继电器执行解锁操作

五、底座电机控制（右侧面板）

此区域提供Modbus电机控制面板，用于控制底座电机的运动。所有功能均为内置实现，不调用外部脚本。

1. 连接并使能
- 按钮名称：连接并使能
- 对应代码：MotorControlPanel.connect_motor() 方法
- 功能说明：连接Modbus电机驱动器并使其能，连接成功后自动执行回零操作

2. 执行移动
- 按钮名称：执行移动
- 对应代码：MotorControlPanel.move_motor() 方法
- 功能说明：根据输入框中的位置值，控制电机移动到指定位置（需要先连接并使能）

3. 回零
- 按钮名称：回零
- 对应代码：MotorControlPanel.move_home() 方法
- 功能说明：控制电机回到零点位置

4. 固定位置按钮
- 按钮名称：100k、170k、200k、300k、318k
- 对应代码：MotorControlPanel.move_fixed() 方法
- 功能说明：快速移动到预设的固定位置（单位：脉冲数）

5. 急停
- 按钮名称：急停
- 对应代码：MotorControlPanel.emergency_stop() 方法
- 功能说明：紧急停止电机运动

六、机械臂配置（右侧面板）

此区域用于控制机械臂底盘的位置配置。所有功能均为内置函数实现，不调用外部脚本。

1. 底盘0位
- 按钮名称：底盘0位
- 对应函数：baseto0()
- 功能说明：将机械臂底盘移动到0号位置

2. 底盘1位
- 按钮名称：底盘1位
- 对应函数：baseF0ToW1()
- 功能说明：将机械臂底盘从0位移动到1号位置

3. 底盘2位
- 按钮名称：底盘2位
- 对应函数：baseFW1ToW2()
- 功能说明：将机械臂底盘从1位移动到2号位置

4. 底盘2-0位
- 按钮名称：底盘2-0位
- 对应函数：baseFW2To0()
- 功能说明：将机械臂底盘从2位移动回0号位置


系统初始化

系统启动时会自动执行以下初始化操作：

1. 初始化双机械臂控制器（Robot1和Robot2）
2. 启动机器人服务器（RobotServer）
3. 初始化深度相机（RealSense）
4. 启动Socket服务器用于图像数据传输

如果初始化失败，状态栏会显示相应的错误信息。
